## SQL注入
### 判断能否sql注入
* 提交单引号 ```id=1'```
* and大法和or大法 
```
id=2 and 1=1
id=2 or 1=1 
```
* 加减法
```
id=1+1
```

### 显错注入和盲注
*  显错注入，即带回显错误提示，可利用联合查询union select来爆出更多信息。
*  盲注，没有任何信息返回，一般可以利用时间sleep()来判断查询成功与否，使用and来盲猜数据库名，表名
