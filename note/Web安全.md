### 内容安全策略（CSP）
内容安全策略   (CSP) 是一个额外的安全层，用于检测并削弱某些特定类型的攻击，包括跨站脚本 (XSS) 和数据注入攻击等

### Web浏览器安全策略
* CSP
* 安全沙箱 H5新增特性，使用标签``<iframe>``
* 同源策略 最基本的安全策略，由网景提出，同源指的是域名、协议、端口相同


## SQL注入
### 判断sql注入
* 提交单引号 ```id=1'```
* and大法和or大法 
```
id=2 and 1=1
id=2 or 1=1 
```
* 加减法
```
id=1+1
```

### GET注入和POST注入
* 注入方式不同，get使用url参数提交注入语言，post通过表格提交

### 显错注入和盲注
*  显错注入，主要是利用了联合查询union select
*  盲注，使用and来盲猜数据库名，表名
